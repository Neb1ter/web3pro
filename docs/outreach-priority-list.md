# Get8 Pro 外链与品牌提及优先级清单

以下顺序按“更适合 Get8 Pro 当前站点定位、更容易长期被收录、更不容易触发垃圾外链判断”排序。

## P1: 最值得优先拿下

1. 品牌自有资料页  
   包括 GitHub、X、Telegram 频道说明、Discord 服务器简介、Medium/Mirror 作者页。目标是统一品牌名、站点定位、主域名和联系入口。
2. 合作交易所或合作方公开介绍页  
   只要能拿到公开合作介绍、活动页、社区合作页，这类链接最能提升真实度。
3. 中文 Web3/加密导航站收录页  
   适合收录你的“学习导航、工具合集、KYC 指南、交易所信息”定位。
4. 工具/资源聚合站的资源页  
   例如“Web3 学习资源”“Crypto Tools Directory”这类页面，更愿意引用有结构的工具合集。
5. 行业作者或研究员引用型链接  
   最适合你的是“交易所对比表”“KYC 常见退回原因”“返佣规则说明”这类可引用资料。
6. Reddit 高质量主题帖  
   用问题切入，再给出结构化答案，比纯推广帖更容易长期有搜索流量。
7. Medium 长文  
   适合发“新手路径”“KYC 流程”“交易所选择方法”这类 evergreen 内容。
8. Mirror 长文  
   更适合偏 Web3 原生读者，发教育型或行业观察型内容。

## P2: 强相关且适合持续做

9. 知乎专栏  
   适合中文搜索和长期索引，尤其适合“新手如何开始”“交易所怎么选”“返佣规则怎么理解”。
10. 掘金 / 少数派 / InfoQ 类知识社区  
    适合偏工具、方法论、导航资源型内容。
11. Substack / Newsletter 资料页  
    如果以后做周报，这类提及能稳定积累品牌检索。
12. 播客 / AMA / 社群分享回顾页  
    不是每个都带链接，但对品牌可信度很有帮助。
13. Web3 教育社群资源贴  
    比如课程群、学习社群、DAO onboarding 资料贴。
14. 项目方或 KOL 的资源推荐贴  
    重点争取引用你的表格或流程图，而不是硬塞首页链接。
15. 问答平台的高质量回答  
    如知乎问答、Quora、Reddit Answers。重在回答问题，链接只做补充。

## P3: 可以做，但别过量

16. 独立博客 guest post  
    只投给真正相关的区块链、工具、教育类博客，不要批量群发。
17. 行业 newsletter 的资源推荐栏目  
    适合投“本周值得收藏的工具/指南”。
18. 开源项目 README 的资源区  
    只在非常贴合时出现，例如链上分析或教育导航相关项目。
19. 站长导航 / 收藏夹类站点  
    要挑质量好的，避免低质目录站。
20. 本地化社区论坛或区域 Telegram 置顶资源贴  
    适合带来真实用户，但最好配合一份高质量说明页。

## 获取外链时的执行原则

- 优先拿品牌词链接，例如 `Get8 Pro`，不要堆“交易所返佣”“币安返佣”这类硬锚文本。
- 优先推可引用内容页，而不是所有地方都推首页。
- 先做 3 类最容易被长期收录的内容页：
  - 交易所对比表
  - KYC 流程与退回原因
  - 币圈工具合集
- 每篇对外内容都要有：
  - 结论
  - 条件
  - 限制
  - 风险提示
  - 可以继续阅读的站内页
- 不买垃圾外链，不做批量目录站群发，不做关键词锚文本刷链。
